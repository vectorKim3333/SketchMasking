🎨 SketchMasking 개인정보 처리방침

핵심 요약: SketchMasking 확장프로그램은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 모든 작업은 로컬 브라우저에서만 처리됩니다.

최종 업데이트: 2025년 9월 28일

1. 개인정보 처리방침 개요

SketchMasking("확장프로그램", "우리")은 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 SketchMasking Chrome 확장프로그램 사용 시 개인정보 처리에 관한 사항을 설명합니다.

2. 수집하는 정보

🚫 수집하지 않는 정보

  • 개인 식별 정보: 이름, 이메일, 전화번호, 주소 등
  • 브라우징 데이터: 방문한 웹사이트, 브라우징 히스토리
  • 웹사이트 콘텐츠: 웹페이지 내용, 사용자 입력 데이터
  • 추적 데이터: 쿠키, 사용자 행동 패턴
  • 민감한 정보: 비밀번호, 카드 정보, 개인 문서

3. 데이터 처리 방식

3.1 로컬 처리

  • 모든 그리기 작업은 브라우저 메모리에서만 처리됩니다
  • 텍스트 마스킹은 현재 페이지에서만 임시적으로 수행됩니다
  • Canvas API를 사용한 오버레이 방식으로 원본 페이지를 수정하지 않습니다

3.2 데이터 저장

중요: SketchMasking은 어떠한 데이터도 영구적으로 저장하지 않습니다.
  • 로컬 저장소 사용 없음
  • 쿠키 설정 없음
  • 브라우저 히스토리 수정 없음
  • 확장프로그램 비활성화 시 모든 변경사항 즉시 제거

4. 네트워크 통신

🌐 외부 통신 없음

SketchMasking은 다음과 같은 네트워크 활동을 수행하지 않습니다:

  • 외부 서버로 데이터 전송
  • 원격 API 호출
  • 분석 데이터 수집
  • 광고 네트워크 연결
  • 타사 서비스와의 통신

5. 권한 사용 목적

5.1 사용 권한 설명

  • activeTab: 현재 활성 탭에서만 기능 실행
  • scripting: 그리기 오버레이 및 마스킹 기능 구현
  • <all_urls>: 모든 웹사이트에서 범용 도구 제공

5.2 권한 사용의 제한

  • 사용자가 직접 확장프로그램을 활성화할 때만 동작
  • 백그라운드에서 자동 실행되지 않음
  • 페이지의 기본 기능을 방해하지 않음

6. 데이터 보안

6.1 보안 조치

  • 로컬 처리: 모든 작업이 사용자 기기에서만 처리됩니다
  • 일시적 저장: 데이터는 브라우저 메모리에만 임시 보관됩니다
  • 즉시 삭제: 기능 비활성화 시 모든 데이터가 즉시 제거됩니다
  • 격리 실행: 다른 확장프로그램이나 웹사이트와 데이터를 공유하지 않습니다

7. 사용자 권리

7.1 제어권

  • 언제든지 확장프로그램을 비활성화할 수 있습니다
  • 그리기 모드를 즉시 종료할 수 있습니다
  • 마스킹된 텍스트를 원래대로 복원할 수 있습니다
  • 확장프로그램 제거 시 모든 흔적이 완전히 제거됩니다

8. 아동 개인정보 보호

SketchMasking은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 실제로 어떠한 개인정보도 수집하지 않으므로 연령에 관계없이 안전하게 사용할 수 있습니다.
