웹페이지 그리기 & 텍스트 마스킹
🎨 그리기 도구
단축키: Alt+1 (맥: Option+1)
🔒 텍스트 마스킹
단축키: Alt+2 (맥: Option+2)
🔍 영역 마스킹
단축키: Alt+3 (맥: Option+3)
📋 사용 방법
🎨 그리기 기능:
  • 활성화: Alt+1 (맥: Option+1)
  • 도구 선택: 박스(▢), 원(○), 펜(✎), 선(/)
  • 그리기: 마우스로 드래그하여 그리기
  • 종료: 도구바의 ✕ 버튼 또는 단축키
🔒 텍스트 마스킹:
  • 마스킹: 텍스트 선택 → Alt+2로 ***로 변환
  • 해제: 선택 없이 Alt+2로 원본 복구
  • 자동 해제: 그리기 모드 종료 시 모든 마스킹 해제
🔍 영역 마스킹:
  • 활성화: Alt+3 (맥: Option+3)
  • 영역 선택: 마우스로 박스 드래그하여 영역 선택
  • 블러 적용: 선택한 영역에 자동으로 블러 마스킹 적용
  • 다중 선택: 여러 영역을 연속해서 마스킹 가능
  • 종료: Alt+3으로 모드 해제 및 마스킹 초기화
💡 팁:
  • 마스킹은 여러 개 영역에 적용 가능
  • 그리기 모드를 끄면 캔버스와 마스킹 모두 초기화
  • 영역 마스킹 시 작은 영역은 무시됩니다
  • 영역 마스킹과 그리기 모드는 동시에 사용 불가
